# NZMAI Level 6 Portfolio

This project is a full stack SaaS platform built during my Level 6 industry project.

It was designed to support mortgage advisers by providing secure workflows, AI-powered tools, and subscription-based access to business features.

## 🚀 Key Features

- Secure authentication and role-based access
- Stripe subscription workflows (checkout, webhooks, billing)
- AI-powered policy search using embeddings and semantic retrieval
- Document upload and validation workflows
- Protected calculator access for subscribed users
- Cloud deployment using Vercel

---

## 🧠 What Problem It Solves

Mortgage advisers needed:
- A secure place to upload and manage documents  
- Faster access to lender policy information  
- Reliable calculator tools  
- A single platform to manage workflows  

This platform centralises all of those into one system.
